      Windows端

Windows端需要建立一个用来进行文件分享的文件夹，比如我在D盘建了一个ShareWithVM文件夹。在这个文件夹的图标上用鼠标右键单击，然后选择属性–>分享，然后选择“高级分享（Advanced Sharing）”并勾选“分享这个文件夹”，然后单击“确定”即可。

Linux端
